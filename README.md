## Specs
1. If number > 3999 , or not a number
	input: 4000 or apple
	output: false/error
2. if number === 1, 5, 10, 50, 100, 500, 1000
	input: 1, 5, 10, 50, 100, 500, 1000
	output: I, V, X, L, C, D, M
3. if number 1-9
	input: 1-9
	output: I-IX
4. if number ===4
	input: 4
	output: IV
5. if number ===9
	input: 9
	output: IX
6. if number 10-99
	input: 10-99
	output: X - XCIX
7. if number 100 - 3999
  input: 100 - 3999
	output: C - MMMCMXCIX
